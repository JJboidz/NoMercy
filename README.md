## 🛡️ About

NoMercy Toolkit is designed for:

- Security researchers  
- Ethical hackers  
- Network administrators  

It provides a variety of attacks and exploits for testing the security posture of networks, applications, and systems in a controlled, ethical environment.  

---

## ⚙️ Features

- Multiple attack vectors including:  
  - Brute-force attacks  
  - Network scanning  
  - Vulnerability scanning  
  - Phishing
  - And many more…    
- Easy-to-read command-line interface  
- Modular design for easy extension
